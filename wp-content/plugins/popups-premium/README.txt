Popups Premium version for the best Wordpress Popups Plugin

== Description ==

With Popups Premium you will take the best free WordPress Popups plugin ever to the next level.

Popups Premium is the perfect solution to gain more subscribers into your mailing lists or to get more Facebook likes, Twitter Followers or Google + Followers.

= Available features =

* Track impressions and conversion
* Integrate it with you google analytics
* Support for CF7, gravity forms and generic forms
* 8 New animations effects (modern browsers only)
* Exit Intent technology
* New trigger methods
* Timer for auto closing
* Ability to disable close button
* Ability to disable Advanced close methods

== Installation ==

1. Unzip and Upload the directory 'popups-premium' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to Popups menu and add as many popups as needed
